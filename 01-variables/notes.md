# Variables

## What is a Variable?

A variable is a named storage for data.

Example:

let message = "Hello";
message = "World";

When the value is changed, the old data is removed from the variable.

## Variable Naming Rules

- Letters, digits, _ and $
- Cannot start with a digit

Examples:

let userName;
let test123;

## Case Sensitive

apple and APPLE are different variables.

## Reserved Names

Cannot use:

- let
- class
- return
- function

## Constants

const birthday = "10.02.2003";

Constants cannot be reassigned.

## Summary

- let → modern declaration
- var → old declaration
- const → fixed value
